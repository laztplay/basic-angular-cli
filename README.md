# Basic Angular 2 project

## Dependencies
- Angular Cli
- Electron
- RESTful server [https://github.com/siteslave/rest-server](https://github.com/siteslave/rest-server)

# Installation 

```
$ git clone https://github.com/siteslave/basic-angular-cli

$ cd basic-angular-cli

$ npm i

$ ng serve 
```

## For Electron

```
$ npm run electron
```
